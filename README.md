Jogo do Número Secreto

Projeto desenvolvido durante meus estudos de lógica de programação em JavaScript.

Sobre o projeto

O jogo consiste em adivinhar um número secreto gerado aleatoriamente pelo sistema.  
O jogador recebe dicas informando se o número digitado é maior ou menor que o número secreto, até acertar.

Tecnologias utilizadas

- JavaScript
- HTML
- CSS

Conceitos praticados

- Variáveis
- Condicionais (if/else)
- Laços de repetição
- Funções
- Math.random()

Como funciona

1. O sistema gera um número aleatório.
2. O jogador digita um número.
3. O jogo informa se o número é maior ou menor.
4. O jogo continua até o jogador acertar.

Objetivo

Praticar lógica de programação e manipulação de dados com JavaScript.
